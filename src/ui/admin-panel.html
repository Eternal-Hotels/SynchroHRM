What This Interface Covers

Mailbox polling

App-only Microsoft Graph access against the `Inbox` for `auditor@eternalhotels.com` with incremental delta sync.

Parsing behavior

PDF report families are parsed into staging tables and CSVs. XLSX attachments are archived and tracked, but intentionally deferred in v1.

Operational storage

Raw attachments, parsed JSON, SQLite state, and latest CSV exports all stay under the local `storage` directory for auditability.

Next lifecycle step

The generated CSVs are staging outputs for a later NetSuite REST integration, not final NetSuite payloads yet.
