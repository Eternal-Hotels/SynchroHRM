What This Viewer Covers

Read and download

Viewer accounts can inspect property report history, open archived files, and download parsed CSV outputs when they are available.

No admin controls

Mailbox sync, reparsing, user management, sender allowlists, and property edits stay separated behind the admin interface.

Operational storage

Raw attachments, parsed JSON, SQLite state, and latest CSV exports all stay under the local `storage` directory for auditability.

Next lifecycle step

The generated CSVs are property-scoped staging outputs for a later NetSuite REST integration, not final NetSuite payloads yet.
